## 甚麼是 solts?

Slots（插槽）：

-   定義： 插槽是 Vue 中用來將組件內容傳遞到子組件的機制。**它允許你在父組件中插入任意內容，並將這些內容傳遞給子組件進行渲染。**

-   使用方式： 在父組件中使用 <slot> 標籤來標記插槽點，子組件中使用相應的名稱來接收和渲染插槽內容。
